# 🧀 奶酪大盗 Cheese Thief（在线版）

4–8 人的社交推理小游戏：找出偷走奶酪的**大盗**。纯静态网页 + WebRTC 玩家直连（PeerJS），**无后端、无账号**，托管在 GitHub Pages 上。

🎮 **在线试玩**：https://fanyiyang.github.io/cheese-thief/

## 怎么玩

1. 一名玩家点 **创建房间**，拿到一个房间号（如 `CHS-7K2`）。
2. 把房间号发给朋友，其他人填昵称 + 房间号，点 **加入房间**。
3. 满 4 人后房主点 **开始游戏**。每人会拿到一张隐藏身份（🐭 睡鼠 / 🧀 奶酪大盗）和一颗骰子。
4. **夜晚**：大盗偷偷拿走奶酪；睡鼠睡觉。
5. **白天**：大家**开语音**（微信 / Discord / 面对面）自由讨论、推理、诈唬。
6. **投票**：每人指认一名怀疑对象。
   - 投出大盗 → 🐭 睡鼠阵营胜利。
   - 投错（投出睡鼠）→ 🧀 大盗胜利。
   - 平票时按“有效点数”决出被翻牌者：拿到奶酪的大盗点数算 **7（最大）**，所以大盗只要并列最高票就会被翻出来。

> 本游戏只负责保密的游戏机制（发身份、骰子、夜晚、投票、结算）。**讨论请自行开语音**。

## 技术说明

- 纯 `HTML/CSS/JS` 单页，无构建步骤。
- 联网用 [PeerJS](https://peerjs.com/) 玩家直连：**房主的浏览器当“裁判”**，只把每个人自己的身份/骰子私下发给本人，别人的身份不会传到你电脑上。
- 信令走 PeerJS 免费公共服务器；数据点对点传输。

### 本地开发

```bash
# 跑纯逻辑单元测试
npm test

# 本地预览（任选其一）
python3 -m http.server 8091
# 然后浏览器打开 http://localhost:8091
```

### 已知限制（v1）

- 固定 **1 个大盗**，5–8 人局暂无“共犯”和特殊角色。
- 房主掉线则本局结束；中途掉线不支持重连（重开房间即可）。
- 极少数对称 NAT 网络下 P2P 可能连不上，换个网络或重试。
