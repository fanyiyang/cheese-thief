# 🧀 奶酪大盗 Cheese Thief（在线版 · 4 人）

4 人社交推理小游戏：找出偷走奶酪的**大盗**。按官方《奶酪大盗》4 人玩法实现。纯静态网页 + WebRTC 玩家直连（PeerJS），**无后端、无账号**，托管在 GitHub Pages 上。

🎮 **在线试玩**：https://fanyiyang.github.io/cheese-thief/

## 怎么玩

1. 一名玩家点 **创建房间**，拿到房间号（如 `CHS-7K2`）；可在大厅切换 **偷看规则（默认开）**。
2. 把房间号发给朋友，其他人填昵称 + 房间号 **加入房间**。
3. 满 4 人房主点 **开始游戏**：每人 1 张隐藏身份（🐭 睡鼠 / 🧀 奶酪大盗）和 **2 颗骰子**。
4. **选醒来之夜**：点数不同的睡鼠挑一个点数睁眼；大盗两点不同则两晚都睁、相同睁一次。
5. **夜晚**（数 1→6，每晚 10 秒 + 声音）：
   - 轮到你点数的那一晚你会睁眼。**同一晚睁眼的人能互相看到对方睁眼**（头像睁眼）。
   - 大盗每次睁眼都拿走奶酪——**同晚睁眼的人会看到是他拿的**（关键线索）。
   - 偷看开 + 你**独自**睁眼时，可偷看一名玩家的点数。
6. **白天**：开语音（微信 / Discord / 面对面）自由讨论、推理、诈唬。
7. **投票**：每人同时指认一人。**票最多者出局；平票则全部出局**，翻身份牌。
   - 大盗出局 → 🐭 睡鼠胜利；否则 → 🧀 大盗胜利。

> 本游戏只负责保密的游戏机制（发身份、骰子、夜晚、投票、结算）。**讨论请自行开语音**。
> 规则来源：官方《奶酪大盗 Cheese Thief》规则书（4 人变化玩法）。

## 技术说明

- 纯 `HTML/CSS/JS` 单页，无构建步骤。
- 联网用 [PeerJS](https://peerjs.com/) 玩家直连：**房主的浏览器当“裁判”**，只把每个人自己的身份/骰子私下发给本人，别人的身份不会传到你电脑上。
- 信令走 PeerJS 免费公共服务器；数据点对点传输。

### 本地开发

```bash
# 跑纯逻辑单元测试
npm test

# 本地预览（任选其一）
python3 -m http.server 8091
# 然后浏览器打开 http://localhost:8091
```

### 已知限制（v1）

- 目前只做 **4 人局**（官方 4 人变化玩法，无共犯）；5–8 人 + 共犯为后续扩展。
- 房主掉线则本局结束；中途掉线不支持重连（重开房间即可）。
- 极少数对称 NAT 网络下 P2P 可能连不上，换个网络或重试。
